**Search** is Grapevine's ad-hoc research mode — query your indexed coverage without creating a permanent topic monitor.

## Search vs topics

| | **Topic** | **Search** |
| --- | --- | --- |
| Purpose | Ongoing monitoring | One-off investigation |
| Alerts | Yes | No |
| Dashboard | Yes | No |
| History | Retained per plan | Session-based |

Use **topics** for always-on listening. Use **search** when you need to dig into something quickly — a breaking story, a competitor mention, or a historical pattern.

## What you can search

- Mention titles and full text
- Authors and outlets
- Entities and tags
- Date ranges and channels

Results use the same mention detail view as Mentions, with sentiment, source, and collaboration tools.
