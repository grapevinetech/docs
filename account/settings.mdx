## Profile

Update your display name and avatar. Sign out from this tab.

## Workspace

Configure workspace-level settings:

- **Workspace name** — shown to all members
- **Logo** — upload a custom workspace logo
- **Timezone** — affects digest timing and date displays
- **Tags** — create and manage workspace-wide mention tags

## Mobile navigation

Customize which tabs appear in the mobile app bottom navigation (up to a configurable limit). Drag to reorder.

## Tags

Workspace tags let you categorize mentions across topics — "crisis", "product-launch", "competitor", etc. Manage the tag list under **Settings → Workspace**.

<Tip>
  Keep your tag list short and consistent. Tags only help if the whole team uses the same vocabulary.
</Tip>
